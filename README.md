# womens-labor-force-logistic-regression
# Logistic Regression Analysis of Women's Labor Force Participation

This project analyzes the determinants of women's labor force participation using logistic regression.

## Methods
- Data cleaning and preprocessing
- Exploratory data analysis
- Logistic regression modeling
- Hypothesis testing
- Model diagnostics
- Gradient descent implementation

## Key Variables
- Education
- Work experience
- Age
- Young children (<6)
- Household income

## Results
The final model achieved 94.7% classification accuracy and identified education, experience, age, childcare, and income as the strongest predictors.
